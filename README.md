# CAXA 参数化自动化脚本

> 基于 Python + win32com 的 CAXA 电子图板参数化驱动工具  
> 支持标准件一键生成、批量出图、智能纠错与进程守护  
> 适用于机械设计自动化场景，落地“机械逻辑代码化”

## 功能

- **COM驱动**  
  通过 `win32com` 连接 CAXA 应用程序，操作图纸对象。

- **参数化绘图**  
  预置 JSON 参数文件，可绘制圆、矩形、螺栓等常用元素，并自动添加尺寸标注文本，可自定义。

- **批量处理**  
  遍历参数文件夹，逐一生成图纸并保存为 `.exb` 格式。

- **智能纠错**  
  完整的异常捕获、参数校验、重试机制，所有操作记录输出到日志文件，便于排查问题。

- **进程守护**  
  监控 CAXA 进程状态，当软件卡死时自动终止并重新启动，保证长时间批量任务的连续性。

- **模拟运行**  
  即使未安装 CAXA，也可在模拟模式下验证逻辑，控制台会完整输出所有绘图指令。

  运行脚本，当提示“CAXA 未运行”时直接回车，将进入模拟模式。

  所有绘图操作会输出到控制台，不会操作真实 CAXA。

  同样会生成日志文件，便于验证参数解析和批量流程。
- **常见问题**
Q：连接 CAXA 失败，提示“无法连接CAXA”
A：检查 CAXA 是否已安装且可正常启动。然后以管理员身份运行脚本。若版本非 2013，可能需要修改 CaxaAutomation 类中的 ProgID，例如 "CAXA.Application.2022"。

Q：绘图效果与预期不符
A：坐标系原点在图纸左下角，单位 mm。调整参数文件中的坐标值即可。
